Accessibility

Our design system meets WCAG 2.1 AA standards for color contrast, keyboard navigation, and screen reader support.

Keyboard Navigation

All interactive elements support keyboard navigation:

  • Tab key navigates through focusable elements
  • Enter/Space activates buttons and links
  • Escape closes modals and dropdowns
  • Arrow keys navigate menus
  • Visible focus indicators (2px outline)

Touch Targets

Minimum touch target size: 44x44px

All interactive elements (buttons, links, form inputs) meet or exceed the minimum 44x44px touch target size recommended by WCAG 2.1 for mobile accessibility.
